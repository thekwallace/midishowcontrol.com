Frequently Asked Questions

How do I select a MIDI output device?

Click the gear icon in the top right corner to open Settings. Under "MIDI Destination," select your output device from the dropdown. Your browser must support Web MIDI (Chrome, Edge, or Opera).

What cue formats are supported?

You can enter a single number for a sequential range (e.g. "10" generates cues 1 through 10), or list specific cue numbers separated by commas (e.g. "1, 5, 10, 15").

What is "Single Cue Mode"?

When enabled, each cue number is treated as an independent GO command. When disabled, cues are sent as a sequential series.

What does "Include GO_OFF" do?

Adding a GO_OFF command at the end of your cue sequence sends a release/stop command after the last cue, which can be useful for automatically ending a sequence on some consoles.

Can I use the exported MIDI file in Pro Tools or another DAW?

Yes. Export your cues as a MIDI file, then import it into your DAW. The file contains standard MIDI Show Control SysEx messages that will be sent through your DAW's MIDI output.

Why can't I see any MIDI devices?

Web MIDI is only supported in Chromium-based browsers (Chrome, Edge, Opera). Safari and Firefox do not support Web MIDI. Make sure your MIDI interface is connected before loading the page. You can also use the refresh button in Settings to re-scan for devices.
